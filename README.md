# Jiffy

Rainmeter skin to display any image, animated or not. Drag n Drop an image into any instance of Jiffy to create a new one.

# Prerequisites

- ffmpeg `winget install Gyan.FFmpeg`

# Features

- Handles .gif images
- Handles animated .webp images
